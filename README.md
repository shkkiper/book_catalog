# Каталог книг

Простой веб-проект с полным CRUD функционалом. Сделан на Python (Flask) и SQLite.

# Возможности

-  Главная страница с карточками книг
-  Детальная страница для каждой книги
-  Открытая админ-панель (без авторизации)
-  Добавление новых книг
-  Редактирование существующих книг
-  Удаление книг
-  Адаптивный дизайн

# Технологии

- Python 3.7+
- Flask (веб-фреймворк)
- SQLite (база данных)

# Как запустить

Сделать имя папки book_catalog
Дважды кликните на файл `start.bat`
После запуска сервера откройте:
Главная страница: http://127.0.0.1:5000
Админ-панель: http://127.0.0.1:5000/admin

# Как пользоваться

Главная страница
Просмотр всех книг в виде карточек
Нажмите "Подробнее" для полной информации

# Админ-панель

Добавить книгу: Заполните форму и нажмите "Добавить книгу"
Редактировать: Нажмите "Редактировать" рядом с любой книгой
Удалить: Нажмите "Удалить" и подтвердите

# Как проверить работу

Запустите сервер
Откройте http://127.0.0.1:5000/admin
Добавьте тестовую книгу:
Название: "Тестовая книга"
Автор: "Тестовый автор"
Год: 2024
Описание: "Это тестовая запись"
Перейдите на главную страницу и убедитесь, что карточка появилась
Нажмите "Подробнее" для просмотра полной информации
Вернитесь в админку и отредактируйте или удалите книгу
